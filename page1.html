$html

格式要求

前端：

1.如果前端需要返回给后台一个数组，比如ids（包含多个id，比如id1和id2），则采用'{"id":["id1","id2"]}'的字符串格式返回;

2.前端对于msg信息的响应，当msg为1，alert一个session已过期的提示，并重定向到主页，当msg为2时，alert一个服务器繁忙的提示

后台：

1.对于每个ajax请求，如果前端有post数据过来，则调用工具函数中ajaxCheck函数来检查请求是否合法

比如：
$currentPage = I('post.currentpage');
if(!ajaxCheck($currentPage)) {
die();
}

2.对于每个数据库操作，都要调用工具函数中queryCheck函数来判断数据库操作是否成功

比如：
$allAlbums = M('album')->select();
if(!queryCheck($allAlbums)) {
die();
}
